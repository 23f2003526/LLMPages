Brandon Sanderson & Data Files Showcase

About This Collection

This site hosts a series of files created as per a professional web development task, including a short story inspired by Brandon Sanderson, ethical dilemma data, SVG art, location recommendations, and forecasts.

Files Available

  • ashravan.txt — A 300-400 word Brandon Sanderson style short story about what happens to Ashravan after Shai restores him.
  • dilemma.json — Ethical scenario data about an autonomous vehicle choosing between hitting 2 people or 1.
  • about.md — Describes myself in exactly three words.
  • pelican.svg — An SVG illustration of a pelican riding a bicycle.
  • restaurant.json — A recommended restaurant location in Chennai with details.
  • prediction.json — Forecast of the Fed Funds rate for December 2025 with reasoning.
  • uid.txt — A unique identifier file uploaded as-is.
  • LICENSE — The MIT license text governing these files.

Usage

Click any file link to view or download the file. These files can be accessed directly from this GitHub Pages deployment.
